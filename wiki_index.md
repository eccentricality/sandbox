## How to use this file
When you are unsure about a mechanic, do not guess. Instead:
1) Identify the system: Alchemy / Stamps / Post Office / etc.
2) Use the lookup items below to find the wiki page(s) that answer:
   - unlock requirements
   - scaling rules
   - best-practice priorities
3) Return advice that is grounded in the wiki.
